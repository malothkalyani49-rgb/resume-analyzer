# AI ATS Resume Analyzer

## 🚀 Overview
This project is an AI-powered ATS Resume Analyzer that evaluates resumes and provides scoring, skill extraction, and job matching insights.

## ⚙️ Features
- Upload Resume (PDF/DOCX)
- ATS Score Calculation
- Skill Extraction
- Job Match Analysis

## 🛠 Tech Stack
- Python
- Streamlit
- Pandas
- Scikit-learn
